Основной шаблон проектирования MVC.
 В классах моделей будут обработчики таблиц: cities, users, files, categories, tasks, replies, opinions, messages, category_user.

 В контроллерах экшены обработки действий на страницах сайта. Классы контроллеров для страниц: Регистрация аккаунта, Вход на сайт, Список заданий, Добавление задания, Список исполнителей, Просмотр задания, Просмотр профиля пользователя, Настройки аккаунта, Мои задания.

 Во вью будет код страницы с выводом информации.

 Вспомогательные классы для действий:
 Класс для обработки и валидации форм.
 Класс для аутентификации.
 Класс для поиска заданий по категориям и городу.
 Класс добавления откликаю.
 Класс для работы с фото для аватарки и фоторо работ.
 Класс для определения роли пользователя и доступных ему действий.
 Класс для обработки переписки заказчика и сисполнителя во внутреннем чате.
 Абстрактный класс дейсвий по заданию, Классы наследующиеся от него по действиям: Отменено, В работе, Выполнено, Провалено.
 Класс на действие выбора исполнителя и назначение его на задание.
 Класс для работы с электронной почтой, получение email уведомлений.
 Класс для обслуживания функций мобильного приложения.
